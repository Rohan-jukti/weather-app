#  Weather App

A simple weather application built using React.js that fetches real-time weather data using OpenWeatherMap API.

## Features
- Search weather by city name
- Shows temperature, humidity, and weather condition
- Clean UI with responsive design

##  Tech Stack
- React.js
- CSS
- OpenWeatherMap API

##  How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
